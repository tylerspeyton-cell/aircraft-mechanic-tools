✅ Aircraft Inspection Checklist

Printable checklist templates for line, daily, and scheduled inspection prep. Use these as working templates, then transfer findings and signoffs to your approved company or aircraft-specific inspection documents.

This page is laid out to print cleanly as a worksheet only. It is not an approved inspection program or task card. Checkbox items are grouped by inspection type so you can use the whole sheet or only the section you need while working from the current approved maintenance data.

Aircraft Inspection Checklist

Turnaround / Preflight

  • Fluid leaks checked at nose, mains, engine, and service panels
  • Tires and brakes visually checked for wear, cuts, and overheating
  • Gear struts / oleos within service limits
  • Cowlings, access doors, and latches secure
  • Static ports, pitot covers, probes, and drains checked
  • Lights and lenses checked for security and condition
  • Control surface freedom and obvious hinge/security issues checked
  • Fuel caps secure and fueling discrepancies noted

Daily / Service Check

  • Engine oil quantity and servicing recorded
  • Hydraulic reservoirs / indicators checked
  • Battery compartment and security checked
  • Wheels, brake lines, and fittings inspected for leakage
  • Landing gear doors, uplocks, downlocks, and rigging visually checked
  • Cabin / baggage / equipment compartments checked for security
  • Avionics cooling inlets / vents clear
  • Outstanding deferred defects reviewed before release

100-Hour / Annual Prep

  • Current AD list and due items pulled
  • Component time / cycle status reviewed
  • Panels removed and tagged as required
  • Compression / borescope / filter inspection tasks prepared if applicable
  • Control cable tension / travel check references available
  • Corrosion-prone areas identified for close inspection
  • Placards, markings, and safety equipment review planned
  • Consumables, seals, cotter pins, and common hardware staged

Signoff / Findings

  • No discrepancies noted
  • Discrepancies entered in work package / logbook
  • Parts required identified
  • Follow-up inspection required

Checklist Build Rules

Keep It Aircraft-Specific

  • Move unique inspection items from the AMM or operator card onto your controlled paperwork
  • Separate visual checks from measured checks to avoid missed signoffs
  • Call out panels, fairings, and access requirements early

Write Actionable Items

  • Use inspect, verify, service, torque, or record instead of vague verbs
  • Reference limits where possible: wear, pressure, fluid level, rigging range
  • Group by zone or system so the mechanic can work in sequence

Close the Loop

  • Every discrepancy needs a location, description, and follow-up path
  • Use a single final signoff area so release status is clear
  • Transfer temporary worksheet notes into the permanent record promptly
